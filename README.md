# 🏠 Interio — Смарт-квиз для дизайн-проекта

> Интерактивная платформа для создания персонального дизайн-проекта интерьера.

## 🚀 Быстрый старт

```bash
# 1. Клонировать
git clone https://github.com/Pavel1778/flutter-cloud-ide.git
cd flutter-cloud-ide

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Запустить
python server.py
```

|----------|-----|

| Квиз | http://localhost:5000/quiz |
| Портфолио | http://localhost:5000/portfolio |
| API Docs | http://localhost:5000/docs |

## 🤖 Telegram-бот

Бот `@mystylequizybot` работает через FastAPI Web Service.

**Команды:**
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие + главное меню |
| `/quiz` | Начать квиз в Telegram |
| `/support` | ИИ-советник (GigaChat) |
| `/requests` | Мои заявки |
| `/about` | О сервисе |
| `/help` | Справка |

**Деплой бота (Render):**
1. New + → Web Service → подключить репозиторий
2. Build: `pip install -r bot/requirements.txt`
3. Start: `uvicorn bot.main:app --host 0.0.0.0 --port $PORT`
4. Env: `TELEGRAM_BOT_TOKEN`, `FRONTEND_URL`

---

## ♿ Доступность

5 режимов с сохранением в localStorage:
- **Высокий контраст** — чёрный фон, белый текст
- **Крупные кнопки** — увеличенные элементы (min 56px)
- **Увеличенный текст** — шрифт 18px+
- **Без анимаций** — отключение transition/animation
- **Подсветка фокуса** — visible outline при Tab

## 🌐 Деплой

### Render (бэкенд + бот)
```yaml
# render.yaml
services:
  - type: web
    name: interio-backend
    env: python
    buildCommand: pip install -r requirements.txt
    startCommand: uvicorn server:app --host 0.0.0.0 --port $PORT
```

### Netlify / Vercel (фронтенд)
```toml
# netlify.toml
[build]
  publish = "templates"

[[redirects]]
  from = "/api/*"
  to = "https://interio-backend.onrender.com/api/:splat"
  status = 200
```
